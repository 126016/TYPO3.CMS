Core update

The install tool can automatically update the TYPO3 CMS core to its latest minor release if certain criteria are met.

End of life reached
TYPO3 CMS 6.2 core version reached end of life and no longer receives regular maintenance releases from the core team. Upgrade to TYPO3 CMS v7 or TYPO3 CMS v8. Another option is the extended Long-Term-Support plan offered by TYPO3 GmbH until March 31st, 2019. See extended support for details.
Disabled
This feature is disabled in this installation (through the environment variable TYPO3_DISABLE_CORE_UPDATER=1)
